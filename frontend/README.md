# Timetable Generator Frontend

React + TypeScript + Vite frontend for the Timetable Generator system.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Opens at `http://localhost:5173`

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:8000
```

## Features

- **Dashboard**: Overview of all entities
- **CRUD Pages**: Manage courses, classes, faculties, lessons, classrooms
- **Timetable Generator**: Automatic timetable generation with visual grid display
- **Responsive Design**: Works on desktop and tablets
- **Color Coding**: Visual differentiation of entities
